一、Lovart 是什么

1. 用产品经理能听懂的话解释

Lovart 是一个 AI 设计智能体。普通设计工具像“工具箱”，你需要自己知道用哪把工具、怎么排版、怎么修图。Lovart 更像“设计执行同事”：你说明业务目标，它自动选择模型、生成素材、放到画布上，再支持你用自然语言继续修改。

举个电商例子：你不是只输入“生成一张护肤品海报”，而是告诉它“这是一个敏感肌修护精华，新品首发，要做小红书风格社媒广告，强调温和、屏障修护、首单 8 折，品牌色是薄荷绿和米白，商品瓶身不能变形”。Lovart 会围绕这个目标产出设计。

2. 它和普通 AI 画图、Canva、Figma、Midjourney 的区别

工具类型像什么适合不适合
LovartAI 设计师 + 画布 + 编辑器从 Brief 到多张电商资产的一体化流程需要严格像素级人工排版的最终精修
Midjourney / 普通画图工具视觉灵感生成器风格探索、氛围图、概念图多素材管理、商品真实性、局部编辑闭环
Canva模板化排版工具套模板、快速排版、轻量运营图从零生成原创商品场景和多轮 AI 编辑
Figma专业协作设计工具精细排版、设计系统、页面交付自动生成商品摄影和广告创意

3. 适合与不适合

适合

  • 需要快速做 5-20 个广告图方向。
  • 需要把一个商品图换成多个生活方式场景。
  • 需要给多个 SKU 做统一风格的商品图。
  • 需要先生成可讨论的视觉方向，再进 Figma / PS 精修。

不适合直接替代

  • 涉及法律强合规的商品说明图，必须人工复核。
  • 需要 100% 还原实物细节、材质、颜色的最终商用图。
  • 中文长文案密集排版，仍建议后续在 Figma / PS / Canva 复排。
  • 平台规则变化频繁，最终尺寸和规范要以平台后台为准。

二、Lovart 的核心工作方式

1. 基本工作流

  1. 准备：上传商品图、Logo、品牌色、字体、参考图、竞品截图、卖点文案。
  2. 选择入口：常见任务先用 Skills；明确知道要什么图时用 Image Generator；复杂项目用 Thinking Mode。
  3. 生成首版：先要 3-5 个方向，不要第一版就追求最终稿。
  4. 局部修改：用 Touch Edit / Mark 点选局部，用 Quick Edit 快速换背景、改颜色、调光影。
  5. 结构调整：用 Edit Elements 拆层移动，用 Text / Edit Text 改文字，用 Frame 组织多尺寸画板。
  6. 适配导出：用 Expand 改比例，用 Crop 裁切，用 Upscale 放大，用 PNG / JPG / PSD / PPTX / SVG 等格式导出。

2. 关键概念

Canvas 画布

像一张无限白板。你可以把主图、Banner、社媒图、竞品参考、修改版本都放在一起对比。

Skills 技能

像预设好的工作流。电商任务优先找 E-Commerce、Marketing、Social Media、Branding 类 Skill。

Brand Kit 品牌包

存 Logo、颜色、字体、设计规则。适合多 SKU、多活动、多平台保持一致。

@ Mention

在输入框用 @ 指定品牌包、画布元素或模型，优先级高于普通偏好。

Thinking / Fast Mode

Thinking 适合复杂品牌和整套活动；Fast 适合先快速出多个方向。

References 参考素材

上传图片、视频、PDF，或给公共链接，让 Lovart 理解风格、商品、竞品和页面结构。

四、核心场景教程

场景 1：商品主图

业务目标让用户在搜索结果页一眼看清商品，并建立真实可信感。

准备材料：高清商品图、平台规范、商品尺寸/颜色、是否允许道具、是否必须白底。

  1. 上传真实商品图，优先用正面、清晰、无遮挡素材。
  2. 如果是平台主图，提示词里明确“不要改变商品结构、Logo、颜色、包装信息”。
  3. 用 Remove BG 去背景，必要时生成纯白背景或浅灰背景。
  4. 用 Crop 控制商品占画面比例；用 Upscale 输出高清版本。
请基于我上传的真实商品图，生成一张电商平台商品主图。要求：纯白背景，商品居中，商品占画面约 80%-85%，保留真实外观、Logo、包装文字、颜色和结构，不添加人物、道具、水印、促销文案或额外图标。光线干净、边缘清晰、适合商品列表页展示。输出 1:1 方图。
Create an e-commerce main product image based on my uploaded real product photo. Use a pure white background, center the product, keep the product filling about 80%-85% of the frame, and preserve the exact product shape, logo, packaging text, colors, and structure. Do not add people, props, watermarks, promotional text, or extra icons. Clean lighting, sharp edges, suitable for a marketplace listing. Output in 1:1 square format.

场景 2：商品场景图 / 生活方式图

业务目标让用户想象“我用它会是什么样”，提升点击和停留。

  1. 上传商品图和目标人群描述。
  2. 先生成 3 个场景方向：家庭、户外、办公、旅行、节日等。
  3. 选择最贴近品牌的一版，用 Touch Edit 调整背景、光影、道具。
  4. 对需要多角度展示的商品，尝试 Multi-Angles，但最终要人工确认商品结构是否被改动。
请把我上传的商品放入一个真实、有购买欲的生活方式场景中。商品必须保持真实外观和比例，不要改变瓶身/包装/Logo。目标用户是 [目标人群]，使用场景是 [场景]。视觉风格：[如高级、清爽、温暖、科技感]。构图要求：商品是视觉中心，背景辅助表达卖点，不要喧宾夺主。输出 4:5 竖版图片，适合小红书/Instagram 商品种草。

场景 3：促销 Banner

业务目标在首页、活动页或店铺头图里快速表达优惠、品类和行动按钮。

  1. 先明确活动：新品首发、限时折扣、满减、节日礼赠。
  2. 用 Frame 建立目标尺寸，例如 1920×600、1200×400、1000×163。
  3. 生成首版时要求保留文案区域，避免文字压在商品关键部位。
  4. 中文促销文案较长时，建议 Lovart 先做视觉版，再在 Figma / Canva 精排文字。
为 [品牌名] 的 [商品/品类] 设计一张电商促销 Banner。活动主题：[活动主题]。主标题：[主标题]，副标题：[副标题]，行动按钮文案：[按钮文案]。视觉风格：[品牌风格]，使用品牌色 [颜色]。构图：左侧/中间展示商品，右侧保留清晰文案区；层级清楚，适合快速扫读。不要改变商品外观，不要生成多余 Logo。尺寸：[填写尺寸，如 1920x600]。

场景 4：社媒广告图

业务目标让用户在信息流里停下来，并在 1-2 秒内知道“卖什么、为什么值得点”。

  1. 先要求 4 个不同创意方向：痛点型、利益型、场景型、对比型。
  2. 每个方向保持同一个 Brand Kit，便于 A/B 测试。
  3. 用 Fast Mode 快速出变化；确定方向后再用 Thinking Mode 做精修逻辑。
请为 [商品名] 生成 4 张社媒广告图，用于 [平台，如 Facebook/Instagram/小红书/TikTok 封面]。目标用户：[目标人群]。核心卖点：[卖点]。请分别做：痛点型、利益型、生活方式型、限时优惠型。每张图都要保留商品真实外观，使用品牌色，文案少而有冲击力。输出尺寸：[1080x1080 或 1080x1920]。

场景 5：商品详情页模块 / A+ 内容

业务目标不是“好看”，而是把用户购买前的疑问逐个讲清楚。

模块作用适合内容
首屏价值图说明这是啥、核心价值是什么商品 + 主卖点 + 使用场景
功能拆解图解释为什么好用3-5 个功能点、局部放大
材质/工艺图建立品质信任材质纹理、结构剖面、质感特写
对比图降低选择成本旧款 vs 新款、竞品 vs 自家
场景故事图让用户代入生活家庭、办公、旅行、节日礼赠
请为 [商品名] 设计一组电商详情页视觉模块，共 5 张。目标是解释商品卖点并提升购买信任。模块包括：1）首屏价值图，2）核心功能拆解图，3）材质/工艺特写图，4）使用场景图，5）对比/规格说明图。请保持品牌风格统一，商品外观真实，不夸大功效。每张图请预留中文标题和短文案位置，尺寸 [例如 970x600 或 750x1000]。

场景 6：品牌视觉套系

业务目标新品或新店铺需要一套统一视觉，而不是每张图各自为战。

  1. 先建 Brand Kit：Logo、品牌色、字体倾向、摄影风格、构图规则、禁用项。
  2. 要求 Lovart 生成“主图、场景图、Banner、社媒图、详情页模块”一套。
  3. 选中最稳定的一套，把成功提示词保存为 Custom Skill。
请基于 @我的Brand Kit，为 [品牌名] 的 [商品系列] 创建一套电商视觉系统。需要包含：商品主图风格、生活方式场景图风格、促销 Banner 风格、社媒广告图风格、详情页模块风格。请输出 3 个视觉方向，每个方向说明适合的人群、色彩、构图、字体气质和可复用规则。

场景 7：节日大促物料

业务目标短时间内覆盖多个渠道，同时保持活动主视觉一致。

为 [品牌名] 的 [节日/大促，如 618、双 11、黑五、圣诞、新年] 设计一套电商活动视觉。活动机制：[折扣/满减/赠品]。目标商品：[商品列表]。请生成：首页主 Banner、商品集合页头图、3 张社媒预热图、1 张倒计时图、1 张邮件头图。整体风格统一，突出节日氛围但不要遮挡商品。请给出每张图的推荐尺寸和文案层级。

场景 8：多平台尺寸改版

业务目标同一个创意适配不同平台，不重复从零设计。

渠道常用尺寸参考注意
商品主图1:1，如 1000×1000、1500×1500以平台最新规范为准，避免多余文字和道具
社媒 Feed1080×1080、1080×1350中心信息清晰，标题不要太小
Story / Reels 封面1080×1920上下留安全区，避免按钮遮挡

五、Lovart 电商 Prompt 写法

稳定公式：角色 + 商品 + 目标用户 + 使用场景 + 视觉风格 + 构图 + 文案 + 尺寸 + 限制条件。

生活类比：不要只对设计师说“帮我做张好看的图”。你要告诉他“卖给谁、放在哪、强调什么、不能改什么”。

基础模板

你是一名电商视觉设计师。请为 [商品名] 设计 [物料类型]。目标用户是 [人群]，使用场景是 [平台/页面]。核心卖点是 [卖点]。视觉风格为 [风格]，品牌色为 [颜色]。构图要求：[主体位置、文案位置、留白]。文案包括：[标题/副标题/按钮]。尺寸：[尺寸]。限制：保持商品真实外观，不要改变 Logo、包装、颜色和结构，不要添加未提供的认证/功效/价格信息。

高级模板

请先根据我上传的商品图、Brand Kit 和参考图，提出 3 个电商视觉方向。每个方向包含：目标人群、核心情绪、色彩方案、构图逻辑、适合渠道、风险点。等我选择方向后，再基于选中的方向生成一组完整物料：主图、场景图、卖点图、促销 Banner、社媒广告图。所有输出必须保持商品真实性和品牌一致性。

多轮修改模板

请只修改我选中的这张图的 [局部区域/元素]，其他部分保持不变。修改要求：[具体变化]。不要改变商品主体、Logo、包装文字、品牌色和画面比例。请生成 3 个微调版本，分别偏 [方向A]、[方向B]、[方向C]。

反向约束模板

限制条件：不要改变商品结构；不要改变瓶身/包装/Logo；不要虚构认证、奖项、功效和价格；不要生成错误中文；不要让文字压住商品；不要添加水印；不要把商品比例拉伸；不要生成与品牌调性冲突的背景；不要让人物手部遮挡商品关键卖点。

六、完整案例

案例 1：美妆护肤品

设计目标：为敏感肌修护精华做新品首发视觉。

品牌设定：温和、可信、轻医研；品牌色薄荷绿、米白、浅银。

请为一款敏感肌屏障修护精华设计新品首发电商视觉。目标用户是 25-35 岁敏感肌女性，场景是小红书种草和品牌官网。请基于我上传的商品瓶身图，生成 3 个视觉方向：轻医研、自然温和、高级极简。每个方向包含商品主图、生活方式场景图、卖点信息图和社媒广告图建议。限制：不要夸大功效，不要改变瓶身、Logo 和包装文字。

修改提示词：“选择轻医研方向。请把背景改成更干净的实验室台面感，增加柔和自然光，减少装饰道具。标题区留在右上角，商品仍居中偏左。”

多尺寸适配：1:1 商品主图、4:5 小红书图、1080×1920 Story、1920×600 官网 Banner。

案例 2：3C 数码产品

设计目标：为无线降噪耳机制作主图、卖点图和黑五广告。

品牌设定：科技、速度、冷静、高性能；主色黑、银、电子蓝。

请基于我上传的无线降噪耳机商品图，设计一套电商视觉。目标用户是通勤白领和游戏用户。核心卖点：主动降噪、低延迟、长续航、舒适佩戴。请输出：1）白底商品主图，2）科技感场景图，3）四个卖点信息图，4）黑五促销广告图。风格：深色科技感、蓝色光效、画面干净。限制：不要改变耳机形状和 Logo，不要虚构参数。

修改提示词：“请只增强蓝色光效和金属质感，不要让背景过暗。卖点图文字区域更清楚，每张只放一个核心卖点。”

交付物：商品主图、降噪场景图、续航图、低延迟图、佩戴舒适图、黑五 Feed 广告、黑五 Story 广告。

案例 3：食品饮料 / 生活方式产品

设计目标：为低糖气泡水做夏季促销和社媒种草。

品牌设定：清爽、年轻、明亮、户外；主色柠檬黄、天空蓝、白色。

请为低糖气泡水品牌设计夏季电商营销视觉。目标用户是 20-30 岁喜欢户外和轻健康饮食的年轻人。商品口味包括柠檬、白桃、青柠。请基于我上传的罐身图，生成：1）三口味组合主图，2）夏日户外场景图，3）低糖卖点图，4）促销 Banner，5）社媒广告图。风格：清爽、明亮、有气泡感。限制：不要改变罐身设计，不要虚构营养成分。

修改提示词：“把夏季氛围增强，但不要变成杂乱海报。减少水果数量，保留气泡和冰块质感，商品罐身更清晰。”

多尺寸适配：店铺横幅、详情页模块、Instagram Feed、小红书封面、邮件头图。

七、使用技巧和避坑

减少 AI 味

  • 用真实商品图做基础，不要只靠文字生成商品。
  • 要求“真实摄影光线、自然阴影、不过度锐化、不过度塑料感”。
  • 一张图只表达一个主卖点，避免堆满元素。

保持商品真实

  • 每次提示都写“不要改变商品结构、Logo、包装文字、颜色”。
  • 重要商品图生成后，对照实物逐项检查。
  • 对透明、反光、金属、小件商品，要多做人工复核。

控制品牌一致性

  • 先建 Brand Kit，再做批量图。
  • 把成功的一组图保存为 Custom Skill，下一批 SKU 复用流程。
  • 同一系列商品用同一光线、背景、构图比例。

处理中文文案

  • 中文短标题可以直接生成，长文案建议后期精排。
  • 把标题、副标题、按钮分层写清楚。
  • 导出前逐字检查错别字、标点、价格和活动规则。

哪些任务在 Lovart 做，哪些任务导出后精修

在 Lovart 内完成导出后建议精修
生成视觉方向、换背景、场景图、广告变化、Mockup、扩图、去背景、局部改图最终中文排版、平台规范复核、法务/功效审查、像素级对齐、复杂表格、印刷前检查

重要：官方和第三方资料都强调 Lovart 能提升速度，但电商图片会直接影响消费者判断。商品颜色、材质、尺寸、功效、认证、活动价格必须人工复核，不能让 AI 自己发挥。

八、实战检查清单

设计前

  • 商品真实图片是否足够清晰
  • 平台尺寸和规则是否确认
  • 品牌色、Logo、字体是否准备好
  • 目标用户和核心卖点是否明确
  • 不能改变和不能宣传的内容是否写清楚

生成中

  • 是否先出多个方向再选一个
  • 是否用 @ 引用 Brand Kit 或商品图
  • 是否每轮只改一个明确问题
  • 是否保留原图和中间版本
  • 是否对比竞品缩略图效果

导出前

  • 商品结构、颜色、Logo 是否准确
  • 中文文案、价格、活动规则是否正确
  • 是否有虚构认证、夸大功效、错误参数
  • 尺寸、清晰度、文件格式是否符合渠道
  • 多 SKU 放在一起是否风格统一
